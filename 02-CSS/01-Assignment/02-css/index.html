Naming Convension in CSS:

    BEM is a convention that attempts to divide the overall user interface into small reusable components.The abbreviation BEM stands for Block‑Element‑Modifier.

        1. Block: encapsulates a standalone entity that is meaningful on its own.

        2. Element: parts of a block that have no standalone meaning.

        3. Modifier: flags on blocks or elements.

        Syntax: block__element--modifier
