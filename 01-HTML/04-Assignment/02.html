HTML Entities:

HTML entities are special codes used in HTML to represent characters that have special meanings in HTML or characters that cannot be easily typed using a keyboard. They are particularly useful when you want to display reserved characters, symbols, or characters from non-Latin scripts in your HTML documents. HTML entities are written using an ampersand (&) followed by a code or name and terminated by a semicolon (;).
Here are five commonly used HTML entities:

  1. & LT;(Less Than) Represents the less than sign (<). It is used to display the character without interpreting it as an HTML tag.

  2. & GT;(Greater Than) Represents the less than sign (>). It is used to display the character without interpreting it as an HTML tag.

  3. & NBSP;(Non-Breaking Sapce) Represents the character which ( ) is invisible without interpreting it as HTML tag.

  4. & COPY;(Copyright) Represents the Copyright symbol (©) without interpreting it as HTML tag.

  5. & REG;(Registered Trademark) Represents the Registered symbol (®) without interpreting it as HTML tag.
